The observed failure of the baseline (Condition A) to learn even at zero latency is directly attributable to the dataset description's constraint regarding the **curriculum latency schedule**. The research plan mandated that the environment be wrapped with a latency-inducing buffer and that the training process include a curriculum where delays are introduced. Because the baseline was trained under this non-Markovian curriculum from the start, the environment's inherent dynamics were fundamentally altered, preventing the agent from ever experiencing a stable, Markovian environment. Consequently, the "baseline" is not a true zero-latency baseline, but rather a model trained on a non-stationary, non-Markovian task, which explains its failure to converge to a viable locomotion strategy.
